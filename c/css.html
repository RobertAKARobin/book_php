You style in a language called CSS, which is completely different from HTML. You can see it in that style="text-decoration:underline;" bit before. CSS can do all kinds of things. It can make elements big, small, blink, shadowed, change color when you hover over them, and much, much more.

Let's say I wanted all of my div elements to be underlined. You've seen one way to do that already:

<div style="text-decoration:underline;">Hamlet's Soliloquoy, by Bill Shakespeare.</div>

<div style="text-decoration:underline;">To be, or not to be: that is the question.</div>

If I have a lot of div elements, writing "text-decoration:underline;" in all of them is going to be a huge pain. Plus, if I decide I don't want them underlined I'd have to go through and delete every occurrence of "text-decoration:underline;"

Fortunately, CSS lets us use stylesheets. ('CSS' stands for Cascading Style Sheets) A stylesheet looks like this:

<style>
div
{
text-decoration:underline;
}
</style>

This takes every div element and underlines it. Now I only need to change one piece of code to change the style of every div.

The div is called a CSS selector because it selects to which elements the CSS rules are being applied. A rule consists of a property and a value. In this case there's only one selector with one rule, but you can have any number. The format's always the same:

<style>
selector1
{
property-a:value-w;
property-b:value-x;
}
selector2
{
property-c:value-y;
property-d:value-z;
}
</style>

You only need two pieces of software to make a website and both of them are already on your computer: a web browser and a plain text editor.

Windows and Mac both come with plain text editors. On Windows it's called Notepad and on Macs it's called TextEdit. They're basically super-simple word processors. Plain text files usually have the file extension .txt.

When you write something in Microsoft Word there's actually lots of hidden code being added to your file. When you write something in plain text what you see on your screen is everything that's in the file. That's why a blank Word file takes up 29 kilobytes of memory and a blank .txt file takes up 0 bytes. That's also why if you open a Word .docx file in Notepad or TextEdit you'll see a huge page of gobbledygook: you're seeing the hidden code.

Your Notepad or TextEdit might be set to work in rich text. Rich text files are like Word files in that they have hidden code, and they usually end in .rtf. To switch to plain text you'll need to look in the settings or preferences for a plain text format option. Usually plain text editors show everything in a monospace font (which means all the letters have the same width) and don't use any bolding or different font sizes.

There are lots of other plain text editors out there, many of them free. A lot of programmers use text editors that include a feature called syntax highlighting. This turns different types of text into different colors so you can tell them apart easily. These colors aren't saved in the files themselves, so the files are still all plain text.

Webpage files end in .html, but otherwise they're just like plain text files. Your index.html should probably be the first page you make. Just create a new document in your text editor and "save" it as index.html.

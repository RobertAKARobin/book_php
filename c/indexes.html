A cookbook is a collection of recipes. To look up a recipe you need to know its page number. Cookbook authors don't expect you to memorize the page number of every single recipe, so they include an index at the end.

A web directory is a collection of files. To look up a file you need to know its URL. Web developers don't expect you to memorize the URL of every single file, so they include a page called index.html in every directory. When someone goes to a directory's URL without specifying a file the server will automatically show them index.html.

Go to amazon.com and you'll see it's exactly the same as amazon.com/index.html. The index is the first site that people will see, so Amazon has worked hard to make it look nice and be good at helping folks get to other pages on amazon.com. If a server can't find an index it'll generate one itself as needed that's just a boring list of URLs.

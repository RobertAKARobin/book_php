You can view a .html file as a webpage in your web browser without being connected to the Internet. Just drag the .html file onto your web browser's icon. Your web browser will open the file, read all its code, and display it as a webpage. This is really handy when you make changes to your code because you can see how they look without having to upload anything to the Internet.

If your file is already open in a web browser, just save your .html file in your plain text editor and then refresh the page in your web browser. The webpage will show any changes you've made.

When you do this, the URL in your address bar will be something like file:///desktop/index.html. The file:/// tells the computer to look inside a local directory called desktop for index.html.

Local means on whatever computer you're using. So if someone on another computer tries to go to file:///desktop/index.html, their computer will look inside itself for a directory called desktop with a file called index.html inside it.

For other people to see your website, you'll need to upload it to your host and they'll need its http:// URL.

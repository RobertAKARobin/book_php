Hosting services are businesses that rent you space on super-fast computers that are made just to run servers. Usually they keep hundreds of these computers in huge underground "server farms."

Hosting services usually sell domains, too. You don't have to change your website's domain to change hosts, you just change your website's DNS settings so nameservers send requests to the new IP address.

Hosting is cheap: I pay about $6 a month. One reason it's so cheap is that my website's uses a shared server with several other people's websites. If I was really paranoid about security, or was expecting my site to get a million views every day, I might want to pay more to use faster computers or use a dedicated server that would serve only my site.

But I'm just an average guy. If I Google "web hosting" anything on the first page of results will probably suit my needs.

Just about every host will automatically set up the www subdomain for you. Other subdomains usually cost extra, and I'd recommend leaving them alone.
